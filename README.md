# About

**Musify** es una aplicación de música similar a Spotify, cuyo objetivo es ser **totalmente gratuita**, sin ningún tipo de anuncios y ofreciendo **libertad total al usuario** sobre lo que decide escuchar.

# Detalles técnicos

Hasta el momento, se encuentran implementadas las siguientes funcionalidades:

- **Splash Screen**
- **3 RecyclerViews**
  - 2 de ellas con filtros.
- **Pantalla de Login**
- **Pantalla de Registro (Register)**
